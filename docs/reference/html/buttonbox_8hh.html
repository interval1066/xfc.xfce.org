Detailed Description

A GtkButtonBox, GtkHButtonBox and GtkVButtonBox C++ wrapper interface.

Provides ButtonBox, HButtonBox and VButtonBox. The primary purpose of ButtonBox is to keep track of the various properties of HButtonBox and VButtonBox. Xfce Foundation Classes
