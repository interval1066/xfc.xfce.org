Detailed Description

A GdkEventSelection C++ wrapper class.

An EventSelection is generated when a selection is requested or ownership of a selection is taken over by another client application.

The event type will be one of SELECTION_CLEAR, SELECTION_NOTIFY or SELECTION_REQUEST.
