Detailed Description

Pango fallback C++ wrapper class.

AttrFallback wraps the fallback attribute (ATTR_FALLBACK).

Constructor & Destructor Documentation

Xfc::Pango::AttrFallback::AttrFallback ( bool  enable_fallback  )  [explicit]

Construct a new font fallback attribute.

Parameters:
enable_fallback Set true if we should fall back on other fonts for characters the active font is missing.
If fallback is disabled, characters will only be used from the closest matching font on the system. No fallback will be done to other fonts on the system that might contain the characters in the text.
