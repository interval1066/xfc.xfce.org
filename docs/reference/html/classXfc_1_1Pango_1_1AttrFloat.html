Detailed Description

A PangoAttrFloat C++ wrapper class.

AttrFloat is the base class for float attributes such as Scale.

Parameters:
attr A float PangoAttribute.
copy Set true if a copy of attr should be used.
If copy is true attr is copied. If false attr is assumed to be a valid float attribute that must be freed. Either way AttrFloat will own the PangoAttribute and free it when it's no longer required.
