Detailed Description

A PangoGlyphVisAttr C++ wrapper class.

GlyphVisAttr is used to communicate information between the shaping phase and the rendering phase. It's contents are still evolving.

Member Function Documentation

bool Xfc::Pango::GlyphVisAttr::is_cluster_start (  )  const

Returns true for the first logical glyph in each cluster.

Clusters are stored in visual order, within the cluster, glyphs are always ordered in logical order, since visual order is meaningless; that is, in Arabic text, accent glyphs follow the glyphs for the base character.
