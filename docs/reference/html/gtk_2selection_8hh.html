Detailed Description

A GtkSelectionData, GtkTargetEntry and GtkTargetList C++ wrapper interface.

The selection mechanism provides the basis for different types of communication between processes. In particular, drag-and-drop and Clipboard work via selections. You will very seldom or never need to use most of the functions in this section directly; Clipboard provides a nicer interface to the same functionality. Some of the datatypes defined this section are used in the Clipboard and drag-and-drop API's as well. Xfce Foundation Classes
