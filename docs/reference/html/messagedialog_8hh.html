Detailed Description

A GtkMessageDialog C++ wrapper interface.

Provides MessageDialog, a dialog with an image representing the type of message (Error, Question, etc.) alongside some message text. It's simply a convenience widget; you could construct the equivalent of MessageDialog from Dialog without too much effort, but MessageDialog saves typing. Xfce Foundation Classes
