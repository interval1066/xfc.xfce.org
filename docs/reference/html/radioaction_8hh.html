Detailed Description

A GtkRadioAction C++ wrapper interface.

Provides RadioAction and RadioActions. RadioAction is an object that represents an action of which only one in a group can be active. RadioActions represents a group of actions. It is a convenience class that is used as a way to easily add a group of radio actions to a Gtk::ActionGroup. Xfce Foundation Classes
