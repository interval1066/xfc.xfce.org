Detailed Description

A GtkTreePath, GtkTreeIter, GtkTreeModel, and GtkTreeRowReference C++ wrapper interface.

Provides TreeIter, TreePath, TreeModel and TreeRowReference, a set of objects that defines a generic tree interface for use by the TreeView widget. It is an abstract interface, and is designed to be usable with any derived class. Xfce Foundation Classes
