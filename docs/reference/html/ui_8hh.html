Detailed Description

A convenient header file that includes the main user interface elements.

This convenient header file includes the header files generally needed by most applications. Note, <xfc/gtk/dialog.hh> includes all the header files in it's inheritance path. These are: <xfc/gtk/window.hh>, <xfc/gtk/bin.hh>, <xfc/gtk/container.hh>, <xfc/gtk/widget.hh>, <xfc/gtk/object.hh> and <xfc/glib/object.hh>. Xfce Foundation Classes
