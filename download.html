You can checkout a working copy the latest XFC source code from the SVN repository:

  • svn co http://svn.xfce.org/svn/xfce/xfc/trunk xfc
Please note that a "working copy" of the XFC source code is quite large because it contains the hidden administrative subdirectories (.svn) Subversion uses to keep the working copy in sync with the repository.

Alternativley, you can download a clean source tree from the SVN repository which is much smaller:

  • svn export http://svn.xfce.org/svn/xfce/xfc/trunk xfc

The SVN repository was last updated May 30th, 2005.
